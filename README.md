# 🏥 Healthcare Mobile App

Cross-platform healthcare application designed to connect patients, doctors, and clinics through a unified mobile platform with scheduling, user management, and API-driven workflows.

------------------------------------------------------------------------

## 🚀 Overview

This project is a real-world healthcare mobile application built to streamline communication and service access between patients and medical providers. The platform supports appointment-related workflows, multi-user interactions, and mobile-first healthcare access.

------------------------------------------------------------------------

## 🧠 Key Contributions

- Developed a cross-platform mobile application using **React Native**
- Integrated a **Laravel/PHP backend** for API-driven workflows
- Implemented features supporting:
  - Patient-doctor interactions
  - Appointment scheduling
  - Clinic and user management
- Collaborated on UI implementation using design workflows from **Figma** and **Adobe XD**
- Built application flows for scalable and user-friendly healthcare access across mobile platforms

## 🧩 System Design

Mobile App (React Native) → REST API Layer (Laravel/PHP) → Database → Healthcare Providers

- API-driven communication between frontend and backend
- Modular architecture for handling users, appointments, and provider workflows
- Separation between UI, business logic, and backend services

------------------------------------------------------------------------

## ✨ Features

- Patient and doctor interaction workflows
- Appointment scheduling and management
- Unified access for clinics and users on a single platform
- Mobile-first healthcare access
- Backend API integration for core medical service workflows
